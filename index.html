SISTEMA DE GESTIÓN SOCIAL

El primer sistema de gestión gratuito para sociedades civiles

Bienvenidos al sistema de gestión de afiliados para Asociaciones civiles.

Creciendo en comunidad

Este sistema ha nacido desde la idea de un miembro de una Asociación Civil que se preguntó: "¿Cómo es que no hay sistemas de gestion gratuitos para pequeñas asociaciones civiles? ¿Si la idea es cambiar el mundo, por que no dejar este aporte a la comunidad?"

Así, una idea se transformó en un proyecto.

Es por esto que el sistema va a estar abierto a las ideas/solicitudes de mejora de la comunidad de usuarios.
Apuntamos a ir creciendo y mejorando en comunidad.

Objetivos

Buscamos solucionar problemas de gestión comunes entre las diferentes organizaciones civiles.
Dentro de las asociaciones civiles, sin fines de lucro, se empiezan a volver tediosas las tareas administrativas.
Cada tarea termina generando una planilla distinta, volviendose todo cada vez más complejo a medida que la estructura de la organización crece.
Dentro del sistema se podrán realizar tareas de:

  • Legajos de miembros
  • Padrón de membrecía
  • Cuotas anuales / mensuales
  • Estados de deudas
  • Libro de caja
  • Extracción de detalle anual de entradas y salidas

¿A quienes está orientado?

Este sistema está orientado a todo tipo de asociaciones civiles y organizaciones sin fines de lucro, como ser:

  • Asociaciones Civiles
  • Grupos Scouts
  • Clubes de barrio
  • Cooperativas
